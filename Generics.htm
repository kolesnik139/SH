Дженериками (обобщёниями) могут быть классы, интерфейсы и методы
Проверка происходит на этапе компиляции!
public <T> T doSomething(T t){ // T Type parameter
	return t;
}

public class Solution <T>{ ..... }

public interface MyGenericInterface<T>
можно реализовать так:
public class MyImplementation implements MyGenericInterface<String>{...}
или так:
public class MyGenericClass<T> implements MyGenericInterface<T>{...}

public class Printer <T extends Animal>{ // bounded generic
	T animal;
	animal.eat() // Common Animal method
}
Для интерфейса тоже extends
public class Printer <T extends Animal & Serrializable & Runnable>{...} // класс первый, за ним интерфейсы
